# SelectList Navigation Wrap-Around & Ultimate Demo Improvements

## Task 1: SelectList Navigation Wrap-Around

**Current Behavior:**
- Pressing UP at the top of the list does nothing
- Pressing DOWN at the bottom of the list does nothing

**Desired Behavior:**
- Pressing UP at the top should wrap to the bottom item
- Pressing DOWN at the bottom should wrap to the top item
- This creates a circular navigation experience

**Implementation:**
Modify `src/pypitui/components.py` in the `SelectList.handle_input()` method:

```python
if matches_key(data, Key.up):
    if self._selected_index > 0:
        self._selected_index -= 1
    else:
        # Wrap around to bottom
        self._selected_index = len(self._filtered_items) - 1
    self._notify_selection_change()
elif matches_key(data, Key.down):
    if self._selected_index < len(self._filtered_items) - 1:
        self._selected_index += 1
    else:
        # Wrap around to top
        self._selected_index = 0
    self._notify_selection_change()
```

**Testing:**
- Test with filtered list (when search is active)
- Test with empty list (should not crash)
- Test with single item (should stay on that item)
